Dialogfeld 'Datei speichern unter'

Mit folgenden Optionen können Sie den Namen und den Speicherort der zu speichernden Datei angeben:

Dateiname

Geben Sie einen Dateinamen an, um ein Dokument unter einem anderen Namen zu speichern. <<YourApp>> fügt die Erweiterung hinzu, die Sie im Feld Datei speichern unter angeben.

Laufwerke

Wählen Sie das Laufwerk aus, auf dem Sie das Dokument speichern möchten.

Verzeichnisse

Wählen Sie das Verzeichnis aus, in dem Sie das Dokument speichern möchten.

Netzwerk

Klicken Sie auf diese Schaltfläche, um die Verbindung mit einem Netzlaufwerk herzustellen, dem ein neuer Laufwerkbuchstabe zugeordnet wird.

<< Fügen Sie weitere Optionen für das Dialogfeld Datei speichern unter hinzu, abhängig davon, welche Optionen Ihre Anwendung über die Flags OFN_ der OPENFILENAME-Struktur auswählt, die von CFileDialog verwendet wird. >>
