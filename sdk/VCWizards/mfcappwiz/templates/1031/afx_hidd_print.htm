Drucken-Dialogfeld

Mit folgenden Optionen können Sie angeben, wie das Dokument gedruckt wird:

Drucker

Zeigt den aktiven Drucker und die Druckerverbindung an. Wählen Sie die Option Setup, um den Drucker und die Druckerverbindung zu ändern.

Setup

Zeigt das Dialogfeld Drucker einrichten zum Auswählen eines Druckers und einer Druckerverbindung an.

Druckbereich

Geben Sie die zu druckenden Seiten an:

Alles Druckt das gesamte Dokument.
Auswahl Druckt den ausgewählten Text.
Seiten Druckt den in den Feldern Von und Bis angegebenen Seitenbereich.

Kopien

Geben Sie die Anzahl der Kopien an, die für den angegebenen Seitenbereich gedruckt werden sollen.

Kopien sortieren

Druckt Kopien in Seitenreihenfolge, anstatt separate Kopien jeder Seite.

Druckqualität

Legt die Druckqualität fest. Eine geringere Qualität beansprucht normalerweise weniger Zeit zum Drucken.
