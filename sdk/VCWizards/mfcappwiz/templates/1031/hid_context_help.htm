Befehl 'Kontexthilfe'

Verwenden Sie diesen Befehl, um die Hilfe für einige Bestandteile von <<YourApp>> anzuzeigen. wenn Sie Kontexthilfe in der Symbolleiste wählen, wird der Mauszeiger zu einem Fragezeichen mit einem Pfeil geändert. Klicken Sie anschließend im <<YourApp>>-Fenster, z.B. auf eine Schaltfläche in der Symbolleiste, um das entsprechende Hilfethema anzuzeigen.
