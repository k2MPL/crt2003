Neu (Menü 'Datei')

Verwenden Sie diesen Befehl, um ein neues Dokument in <<YourApp>> zu erstellen. Wählen Sie den Typ der zu erstellenden Datei im Dialogfeld Neue Datei aus. << Entfernen Sie den vorherigen Satz, wenn Ihre Anwendung nur einen Dokumenttyp unterstützt. >>

Sie können ein vorhandenes Dokument mit dem Befehl Öffnen öffnen.
