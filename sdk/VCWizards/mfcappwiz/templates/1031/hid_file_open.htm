Öffnen (Menü 'Datei')

Verwenden Sie diesen Befehl, um ein vorhandenes Dokument in einem neuen Fenster zu öffnen. Sie können mehrere Dokumente gleichzeitig öffnen. Verwenden Sie das Menü Fenster, um zwischen mehreren geöffneten Fenstern zu wechseln. Siehe Befehl 'Window 1, 2, ...'.

Sie können mit dem Befehl Neu ein neues Dokument erstellen.
