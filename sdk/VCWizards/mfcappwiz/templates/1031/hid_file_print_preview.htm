Seitenansicht (Menü 'Datei')

Verwenden Sie diesen Befehl, um das aktive Dokument so anzuzeigen wie es gedruckt wird. Wenn Sie diesen Befehl wählen wird das Hauptfenster durch eine Seitenansicht ersetzt, in der eine oder zwei Seiten im Druckformat angezeigt werden. Die Symbolleiste Seitenansicht enthält Optionen zum Anzeigen von einer oder zwei Seiten gleichzeitig; zum Blättern durch das Dokument; zum Verkleinern und Vergrößern von Seiten; und zum Initialiseren eines Druckauftrags.
