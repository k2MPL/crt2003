Befehle 'Speichern' (Dateimenü)

Verwenden Sie diesen Befehl, um das aktive Dokument unter den aktuellen Namen und Verzeichnis zu speichern. Wenn das Dokument das erste Mal gespeichert wird, zeigt <<YourApp>> das Dialogfeld Speichern unter an, in dem Sie das Dokument benennen können. Wenn Sie den Namen und das Verzeichnis eines vorhandenen Dokuments ändern möchten, wählen Sie den Befehl Speichern unter.
