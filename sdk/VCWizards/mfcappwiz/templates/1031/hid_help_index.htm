Index (Menü 'Hilfe')

Verwenden Sie diesen Befehl, um das Startfenster der Hilfe anzuzeigen. In diesem Fenster können Sie schrittweise Anweisungen für die Verwendung von <<YourApp>> und verschiedene Referenzinformationen anzeigen.

Wenn die Hilfe geöffnet ist, können Sie jederzeit auf Inhalt klicken, um zum Startfenster zurückzukehren.
