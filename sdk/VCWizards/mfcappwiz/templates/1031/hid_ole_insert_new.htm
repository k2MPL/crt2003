Neues Objekt einfügen (Menü 'Bearbeiten')

Fügt und bettet ein Objekt ein, z.B. ein Diagramm oder eine Formel in einem Dokument. Die Anwendung, in der das Objekt erstellt wurde, wird aktiviert.

Wählen Sie das Objekt aus, das Sie mit dem Dialogfeld Neues Objekt einfügen in das Dokument einfügen möchten.
