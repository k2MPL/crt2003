Symbolleiste(Menü 'Ansicht')

Verwenden Sie diesen Befehl, um die Symbolleiste ein- oder auszublenden. Die Symbolleiste enthält Schaltflächen für die wichtigsten Befehle in <<YourApp>>, z.B. Datei öffnen. Das Menüelement wird markiert, wenn die Symbolleiste angezeigt wird.

Hilfe zur Verwendung der Symbolleiste finden Sie unter Symbolleiste.
