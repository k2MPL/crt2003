Boîte de dialogue Imprimer

Les options suivantes vous permettent de spécifier les paramètres d'impression du document :

Imprimante

Affiche l'imprimante et la connexion d'imprimante actives. Choisissez l'option Configuration pour changer d'imprimante et de connexion d'imprimante.

Configuration

Affiche la boîte de dialogue Configuration de l'impression, qui vous permet de sélectionner une imprimante et une connexion d'imprimante.

Étendue

Spécifie les pages que vous souhaitez imprimer :

Tout Imprime la totalité du document.
Sélection Imprime le texte actuellement sélectionné.
Pages Imprime les pages que vous avez spécifiées dans les champs De et À.

Copies

Spécifie le nombre de copies à imprimer pour l'étendue de pages spécifiée.

Copies assemblées

Imprime les copies dans l'ordre des pages, au lieu d'imprimer toutes les copies d'une même page avant d'imprimer la page suivante.

Qualité d'impression

Spécifie la qualité d'impression. En règle générale, l'impression est d'autant plus rapide que la qualité d'impression choisie est faible.
