Commande Couper (menu Edition)

Utilisez cette commande pour supprimer les données sélectionnées du document et les placer dans le Presse-papiers. Cette commande n'est pas disponible si aucune donnée n'a été préalablement sélectionnée.

Lorsque vous utilisez cette commande, le contenu du Presse-papiers est remplacé par les nouvelles données coupées.
