Commande Nouveau (menu Fichier)

Utilisez cette commande pour créer un nouveau document dans <<YourApp>>. Dans la boîte de dialogue Nouveau fichier, sélectionnez le type de nouveau fichier que vous souhaitez créer. << Supprimez la phrase précédente si votre application ne prend en charge qu'un seul type de document. >>

Vous pouvez ouvrir un document existant à l'aide de la commande Ouvrir.
