Commande Ouvrir (menu Fichier)

Utilisez cette commande pour ouvrir un document existant dans une nouvelle fenêtre. Vous pouvez ouvrir plusieurs documents à la fois. Utilisez le menu Fenêtre pour passer d'un document ouvert à un autre. Consultez Commande 1, 2, ... (menu Fenêtre).

Vous pouvez créer de nouveaux documents à l'aide de la commande Nouveau.
