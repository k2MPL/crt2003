Commande Enregistrer (menu Fichier)

Utilisez cette commande pour enregistrer le document actif sous son nom actuel et dans le répertoire en cours. Si vous enregistrez un document pour la première fois, <<YourApp>> affiche la boîte de dialogue Enregistrer sous qui vous permet d'attribuer un nom à votre document. Si vous souhaitez changer le nom et le répertoire d'un document existant avant de l'enregistrer, choisissez la commande Enregistrer sous.
