Commande Envoyer (menu Fichier)

Utilisez cette commande pour envoyer le document actif par messagerie électronique. Cette commande affiche une fenêtre de messagerie contenant le document actif attaché. Renseignez les champs À, Objet, etc. et ajoutez du texte dans le corps du message, le cas échéant. Lorsque vous avez fini, cliquez sur le bouton Envoyer pour envoyer le message.
