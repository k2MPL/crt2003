Commande Fractionner (menu Fenêtre)

Utilisez cette commande pour fractionner la fenêtre active en volets. Pour déplacer les barres de fractionnement, utilisez la souris ou les touches de direction. Une fois les barres de fractionnement à l'emplacement souhaité, cliquez avec la souris ou appuyez sur ENTRÉE. Si vous appuyez sur ÉCHAP, les barres de fractionnement restent à leur emplacement d'origine. << Dans une application dotée d'une interface monodocument (SDI), cette commande se trouve dans le menu Affichage. >>
