Modification du document

<< Écrivez ici une aide qui explique à l'utilisateur comment modifier un document dans votre application.

Si votre application prend en charge plusieurs types de documents et que vous souhaitez créer une rubrique d'aide distincte pour chacun d'eux, utilisez l'ID de contexte d'aide obtenu en exécutant le fichier MAKEHELP.BAT généré par l'Assistant Application. Vous pouvez aussi exécuter MAKEHM comme suit :

makehm IDR_HIDR_,0x2000 resource.h

Si le symbole IDR_ pour l'un de vos types de documents est IDR_CHARTTYPE, par exemple, l'ID de contexte d'aide généré par MAKEHM est HIDR_CHARTTYPE.

Remarque : L'Assistant Application définit l'ID de contexte d'aide HIDR_DOC1TYPE utilisé par cette rubrique d'aide pour le premier type de document pris en charge par votre application. Cet Assistant crée un alias dans le fichier .HPJ de votre application, qui associe l'ID de contexte d'aide HIDR_DOC1TYPE au symbole HIDR_ généré par MAKEHM pour ce type de document. >>
