Commandes du menu Edition

Le menu Edition propose les commandes suivantes :

[!if CONTAINER || CONTAINER_SERVER] [!endif]
Annuler Annule la dernière opération de modification.
Couper Supprime les données sélectionnées du document et les place dans le Presse-papiers.
Copier Copie les données sélectionnées du document vers le Presse-papiers.
Coller Colle les données contenues dans le Presse-papiers dans le document.
Coller avec liaison Colle à partir du Presse-papiers une liaison à des données d'une autre application.
Insérer un objet Insère et incorpore un objet, tel qu'un graphique ou une équation, dans un document.
Liens Affiche et modifie les liens à des documents incorporés.
