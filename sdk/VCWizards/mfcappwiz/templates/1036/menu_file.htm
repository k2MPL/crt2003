Commandes du menu Fichier

Le menu Fichier propose les commandes suivantes :

[!if PRINTING] [!endif] [!if MAPI] [!endif]
Nouveau Crée un nouveau document.
Ouvrir Ouvre un document existant.
Fermer Ferme un document ouvert.
Enregistrer Enregistre un document ouvert sous le même nom de fichier.
Enregistrer sous Enregistre un document ouvert sous le nom de fichier que vous spécifiez.
Imprimer Imprime un document.
Aperçu avant impression Affiche le document à l'écran tel qu'il apparaîtra une fois imprimé.
Configuration de l'impression Sélectionne une imprimante et une connexion d'imprimante.
Envoyer Envoie le document actif par messagerie électronique.
Quitter Quitte <<YourApp>>.
