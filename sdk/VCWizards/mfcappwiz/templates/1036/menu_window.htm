Commandes du menu Fenêtre

Le menu Fenêtre propose les commandes suivantes, qui vous permettent de réorganiser l'affichage des fenêtres de plusieurs documents dans la fenêtre de l'application :

[!if SPLITTER] [!endif]
Nouvelle fenêtre Crée une nouvelle fenêtre qui affiche le même document.
Cascade Réorganise les fenêtres de sorte qu'elles se superposent.
Mosaïque Réorganise les fenêtres en mosaïque de sorte qu'elles ne se superposent pas.
Réorganiser les icônes Réorganise les icônes des fenêtres fermées.
Fractionner Fractionne la fenêtre active en volets.
Fenêtre 1, 2, ... Rend active la fenêtre spécifiée.
