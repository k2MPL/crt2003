Finestra di dialogo Apri

Le seguenti opzioni consentono di specificare il file da aprire:

Nome file

Specifica il file che si desidera aprire. Questa casella elenca i file con l'estensione selezionata nel campo Tipo file.

Tipo file

Specifica il tipo di file che si desidera aprire:

<< Elencare qui i tipi di file dell'applicazione. >>

Unità

Specifica l'unità in cui <<YourApp>> archivia il file che si desidera aprire.

Directory

Specifica la directory in cui <<YourApp>> archivia il file che si desidera aprire.

Rete

Scegliere questo pulsante per connettersi a un percorso di rete assegnandogli una nuova lettera di unità.
