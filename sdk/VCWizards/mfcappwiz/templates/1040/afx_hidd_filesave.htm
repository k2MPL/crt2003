Finestra di dialogo Salva con nome

Le seguenti opzioni consentono di specificare il nome e il percorso del file che si intende salvare:

Nome file

Specifica un nome file che consente di salvare un documento con un nome differente. <<YourApp>> aggiunge l'estensione specificata nella casella Tipo file.

Unità

Specifica l'unità in cui si desidera archiviare il documento.

Directory

Specifica la directory in cui si desidera archiviare il documento.

Rete

Scegliere questo pulsante per connettersi a un percorso di rete assegnandogli una nuova lettera di unità.
