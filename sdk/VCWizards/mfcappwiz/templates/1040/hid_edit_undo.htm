Comando Annulla/Impossibile annullare (menu Modifica)

<< L'interfaccia dell'applicazione per il comando Annulla potrebbe essere diversa da quella descritta di seguito. Modificare il testo della Guida in base alle differenze. >>

Utilizzare questo comando per annullare l'ultima azione di modifica. Il nome del comando cambia, a seconda dell'ultima azione. Se non è possibile annullare l'ultima azione, il comando Annulla si modifica in Impossibile annullare.
