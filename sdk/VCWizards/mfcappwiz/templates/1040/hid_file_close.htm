Comando Chiudi (menu File)

Utilizzare questo comando per chiudere tutte le finestre contenenti il documento attivo. <<YourApp>> chiede di salvare le modifiche al documento prima di chiuderlo. Se si chiude un documento senza salvarlo, verranno perse le ultime modifiche. Prima di chiudere un documento senza nome, <<YourApp>> visualizza la finestra di dialogo Salva con nome che consente di assegnare un nome e di salvare il documento.
