Comando Salva (menu File)

Utilizzare questo comando per salvare il documento attivo con lo stesso nome e nella stessa directory. Quando si salva un documento per la prima volta, <<YourApp>> visualizza la finestra di dialogo Salva con nome in modo che sia possibile assegnare un nome al documento. Se si desidera modificare il nome e la directory di un documento esistente prima di salvarlo, scegliere il comando Salva con nome.
