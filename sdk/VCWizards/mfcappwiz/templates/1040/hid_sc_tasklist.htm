Comando Passa a (menu di controllo dell'applicazione)

Utilizzare questo comando per visualizzare un elenco di tutte le applicazioni aperte. Utilizzare questo Elenco attività per passare a un'applicazione dell'elenco o per chiudere un'applicazione.

Opzioni della finestra di dialogo

Quando si sceglie il comando Passa a, viene visualizzata una finestra di dialogo con le seguenti opzioni:

Elenco attività

Consente di selezionare l'applicazione a cui si desidera passare o che si desidera chiudere.

Passa a

Rende attiva l'applicazione selezionata.

Termina operazione

Chiude l'applicazione selezionata.

Annulla

Chiude la casella Elenco attività.

Sovrapponi

Dispone le applicazioni aperte in modo che si sovrappongano e sia possibile vedere la barra del titolo di ognuna. Questa opzione non interessa le applicazioni ridotte a icona.

Affianca

Dispone le applicazioni aperte in finestre non sovrapposte. Questa opzione non interessa le applicazioni ridotte a icona.

Disponi icone

Dispone le icone di tutte le applicazioni ridotte a icona nella parte inferiore dello schermo.
