Comando Dividi (menu Finestra)

Utilizzare questo comando per dividere la finestra attiva in riquadri. È quindi possibile utilizzare il mouse o i tasti di direzione per spostare le barre di divisione. Al termine, fare clic sul pulsante del mouse o premere il tasto INVIO per lasciare le barre di divisione nella nuova posizione. Quando si preme ESC, le barre di divisione tornano nelle posizioni originali. << Nelle applicazioni con interfaccia a documento singolo, questo comando appare nel menu Visualizza. >>
