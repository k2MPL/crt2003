Comandi del menu File

Il menu File comprende i seguenti comandi:

[!if PRINTING] [!endif] [!if MAPI] [!endif]
Nuovo Crea un nuovo documento.
Apri Apre un documento esistente.
Chiudi Chiude un documento aperto.
Salva Salva un documento aperto con lo stesso nome file.
Salva con nome Salva un documento aperto con un nome file specificato.
Stampa Stampa un documento.
Anteprima di stampa Visualizza il documento così come apparirà una volta stampato.
Imposta stampante Seleziona una stampante e la connessione alla stampante.
Invia... Invia il documento attivo tramite posta elettronica.
Esci Esce da <<YourApp>>.
