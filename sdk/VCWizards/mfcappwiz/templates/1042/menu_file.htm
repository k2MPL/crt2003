파일 메뉴 명령

편집 메뉴에는 다음과 같은 명령이 있습니다.

[!if PRINTING] [!endif] [!if MAPI] [!endif]
새로 만들기 새 문서를 만듭니다.
열기 기존 문서를 엽니다.
닫기 열린 문서를 닫습니다.
저장 같은 파일 이름을 사용하여 열린 문서를 저장합니다.
다른 이름으로 저장 열린 문서를 지정한 파일 이름으로 저장합니다.
인쇄 문서를 인쇄합니다.
인쇄 미리 보기 인쇄 후 나타날 문서의 모양을 화면에서 표시합니다.
인쇄 설정 프린터 및 프린터 연결을 선택합니다.
보내기... 활성 문서를 전자 우편으로 보냅니다.
끝내기 <<YourApp>>을(를) 끝냅니다.
