Cuadro de diálogo Abrir archivo

Las siguientes opciones permiten especificar el archivo que se desea abrir:

Nombre de archivo

Especifica el archivo que desea abrir. Este cuadro muestra archivos con la extensión seleccionada en el campo Mostrar archivos de tipo.

Mostrar archivos de tipo

Especifica el tipo de archivo que desea abrir:

<< Muestre aquí los tipos de archivo de la aplicación. >>

Unidades

Especifica la unidad en la que <<SuAplicación>> almacena el archivo que desea abrir.

Directorios

Especifica el directorio en el que <<SuAplicación>> almacena el archivo que desea abrir.

Red

Haga clic en este botón para conectarse a una ubicación de red asignándole una nueva letra de unidad.
