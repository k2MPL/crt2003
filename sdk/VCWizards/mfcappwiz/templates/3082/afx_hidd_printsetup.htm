Cuadro de diálogo Configurar impresión

Las siguientes opciones permiten seleccionar la impresora de destino y su conexión.

Impresora

Especifica la impresora que desea usar. Elija la impresora predeterminada o la opción Impresora específica y seleccione una de las impresoras actualmente instaladas que aparecen en el cuadro. Utilice el Panel de control de Windows para instalar impresoras y configurar puertos.

Orientación

Especifica las opciones Vertical u Horizontal.

Tamaño

Especifica el tamaño de papel en que se imprimirá el documento.

Origen

Especifica el origen del papel, ya que algunas impresoras tienen múltiples bandejas para diversos orígenes de papel.

Opciones

Muestra un cuadro de diálogo en el que se pueden seleccionar opciones adicionales de impresión según el tipo de impresora que ha seleccionado.

Red

Haga clic en este botón para conectarse a una ubicación de red asignándole una nueva letra de unidad.
