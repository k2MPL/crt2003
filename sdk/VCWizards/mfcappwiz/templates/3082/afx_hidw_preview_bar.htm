Barra de herramientas de vista preliminar

La barra de herramientas de vista preliminar tiene las siguientes opciones:

Imprimir

Abre el cuadro de diálogo Imprimir para iniciar un trabajo de impresión.

Siguiente

Muestra una vista preliminar de la siguiente página.

Anterior

Muestra una vista preliminar de la página anterior.

Una página / Dos páginas

Muestra una vista preliminar de una o dos páginas impresas a la vez.

Acercar

Amplía la página impresa.

Alejar

Reduce la página impresa.

Cerrar

Cambia del modo de Vista preliminar a la ventana de edición.
