Comando Deshacer o No se puede deshacer (menú Edición)

<< La interfaz de usuario de la aplicación para el comando Deshacer puedeser diferente de la que aquí se describe. Modifique este texto de ayuda según proceda. >>

Utilice este comando para revertir la última acción de edición, si es posible. El nombre del comando cambia según la última acción realizada. En el menú, el comando Deshacer cambia a No se puede deshacer, si no se puede revertir la última acción realizada.
