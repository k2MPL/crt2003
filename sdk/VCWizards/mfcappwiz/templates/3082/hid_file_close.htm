Comando Cerrar (menú Archivo)

Utilice este comando para cerrar todas las ventanas que contienen el documento activo. <<SuAplicación>> le sugiere que guarde los cambios aplicados al documento antes de cerrarlo. Si cierra un documento sin guardarlo, se perderán todos los cambios efectuados desde la última vez que se guardó. Antes de cerrar un documento sin título, <<SuAplicación>> muestra el cuadro de diálogo Guardar como y le sugiere que asigne un nombre al documento y lo guarde.
