Comando Vista preliminar (menú Archivo)

Utilice este comando para ver el documento activo tal como se imprimirá. Cuando se elige este comando, la ventana principal se reemplaza por la ventana de la vista preliminar en la que aparecerán una o dos páginas con el formato de impresión. La barra de herramientas Vista preliminar ofrece opciones para ver una o dos páginas, avanzar o retroceder en el documento, acercar o alejar páginas e iniciar un trabajo de impresión.
