Comando Guardar (menú Archivo)

Utilice este comando para guardar el documento activo con su nombre y en el directorio actuales. Cuando guarda un documento por primera vez <<;SuAplicacióngt;gt; muestra el cuadro de diálogo Guardar como para que pueda asignar un nombre al documento. Si desea cambiar el nombre y el directorio de un documento existente antes de guardarlo, elija el comando Guardar como.
