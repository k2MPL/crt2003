Comando Índice (menú Ayuda)

Utilice este comando para mostrar la pantalla inicial de Ayuda. Desde esta pantalla, puede obtener acceso a instrucciones paso a paso sobre cómo usar <<SuAplicación>> y a diversa información de referencia.

Cuando abra la Ayuda, puede hacer clic en el botón Contenido siempre que desee regresar a la pantalla inicial.
