Comando Cerrar (menú Control)

Utilice este comando para cerrar la ventana o cuadro de diálogo activos.

Hacer doble clic en un cuadro de menú Control equivale a elegir el comando Cerrar.

Nota: si tiene múltiples ventanas abiertas para un mismo documento, el comando Cerrar del menú Control del documento sólo cierra una ventana a la vez. Puede cerrar todas las ventanas al mismo tiempo mediante el comando Cerrar del menú Archivo.
