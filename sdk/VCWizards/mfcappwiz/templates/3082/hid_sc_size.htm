Comando Tamaño (menú del sistema)

Utilice este comando para mostrar una flecha de cuatro puntas que permite cambiar el tamaño de la ventana activa mediante las teclas de dirección.

Cuando el puntero se convierta en la flecha de cuatro puntas:

1.Presione una de las teclas de dirección (teclas de flecha izquierda, derecha, arriba o abajo) para mover el puntero hasta el borde que quiere mover.

2.Presione una tecla de dirección para mover el borde.

3.Presione ENTRAR cuando la ventana tenga el tamaño que desea.

Nota: este comando no está disponible si maximiza la ventana.
