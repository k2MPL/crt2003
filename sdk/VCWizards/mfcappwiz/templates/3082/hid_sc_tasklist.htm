Comando Cambiar a (menú Control de la aplicación)

Utilice este comando para mostrar una lista de todas las aplicaciones abiertas. Use esta Lista de tareas para cambiar a una aplicación de la lista o cerrar dicha aplicación.

Opciones del cuadro de diálogo

Al elegir el comando Cambiar a, aparecerá un cuadro de diálogo con las siguientes opciones:

Lista de tareas

Especifica la aplicación a la que desea cambiar o que desea cerrar.

Cambiar a

Activa la aplicación seleccionada.

Finalizar tarea

Cierra la aplicación seleccionada.

Cancelar

Cierra el cuadro Lista de tareas.

Cascada

Organiza las aplicaciones abiertas de modo que se superpongan y se pueda ver cada barra de título. Esta opción no afecta a las aplicaciones que se han reducido a un icono.

Mosaico

Organiza las aplicaciones abiertas de modo que las ventanas no se superpongan. Esta opción no afecta a las aplicaciones que se han reducido a un icono.

Organizar iconos

Organiza los iconos de todas las aplicaciones que se han minimizado en la parte inferior de la pantalla.
