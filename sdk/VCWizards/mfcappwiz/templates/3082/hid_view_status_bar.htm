Comando Barra de estado (menú Ver)

Utilice este comando para mostrar y ocultar la barra de estado, en la que se describe la acción que realizará el elemento de menú seleccionado o el botón de la barra de herramientas que se ha presionado, así como el estado de bloqueo del teclado. Cuando se muestra la barra de estado, aparece una marca de verificación al lado del elemento de menú.

Consulte el vínculo Barra de estado para obtener ayuda sobre el uso de la barra de estado.
