Comando Barra de herramientas (menú Ver)

Utilice este comando para mostrar y ocultar la barra de herramientas, que tiene botones para algunos de los comandos más usados de <<SuAplicación>>, como el comando Abrir del menú Archivo. Cuando se muestra la barra de herramientas, aparece una marca de verificación al lado del elemento de menú.

Consulte el vínculo Barra de herramientas para obtener ayuda sobre el uso de la barra de herramientas.
