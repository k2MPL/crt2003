Comandos del menú Archivo

El menú Archivo contiene los siguientes comandos:

[!if PRINTING] [!endif] [!if MAPI] [!endif]
Nuevo Crea un nuevo documento.
Abrir Abre un documento existente.
Cerrar Cierra un documento abierto.
Guardar Guarda un documento abierto con el mismo nombre de archivo.
Guardar como Guarda un documento abierto con el nombre de archivo especificado.
Imprimir Imprime un documento.
Vista preliminar Muestra el documento en la pantalla tal como se imprimirá.
Configurar impresión Selecciona una impresora y una conexión de impresora.
Enviar... Envía el documento activo por correo electrónico.
Salir Sale de <<SuAplicación>>.
